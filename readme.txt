Simple Plugin to display a Changelog on an App, Plugin or whatever else page..<br>

== Description ==

With this Plugin you can display project related changelogs on your Wordpress Site.<br>
<br>
This could be very useful, if you have a few projects, which changes you would like to manage on a central place in the Backend of your Wordpress Pages.<br>
<br>
See it in action at: (http://en.kilo-moana.com/changelog/)

== Installation ==

Download plugin through the wordpress included plugin downloader.<br>
Activate plugin in plugin tab.<br>
<br>
Create a Project.<br>
<br>
Create one ore more Versions (Don't forget to choose your Project).<br>
<br>
Place the [changelog project="SLUG OF YOUR PROJECT"] shortcode in one of your posts or pages.
